# **Overview**
The Web Authentication API (WebAuthn) is a web standard that enables passwordless authentication using public-key cryptography. It allows websites to authenticate users using authenticators such as biometrics, hardware security keys, or device-based credentials.
WebAuthn works together with the **FIDO2 authentication standard and is supported by modern browsers.
Key Features:
  ~ Passwordless authentication
  ~ Phishing-resistant login
  ~ Strong cryptographic security
  ~ Works with biometrics (fingerprint, face recognition)
  ~ Supports hardware security keys
